## 적용기술
 React, Hooks, Styled Component, TypeScript, json-server

## 구현내용 
1. 가공방식,재료,상담요청 필터 기능구현
2. 상담요청 토글 버튼 구현
3. 견적 요청카드 hover시 보더 스타일
4. 조건부 렌더링 이용 빈 화면 구현

## 시작 방법

### Installation
```
npm install
```

### Run
```
npm start
```

### Json Sever
```
npx json-server public/data/requests.json --port 4000
```
